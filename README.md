# 🌐 Adrian Cole Portfolio Website

A modern, responsive, and SEO-friendly personal portfolio website built for showcasing professional work, skills, and services. This project is designed with a clean UI/UX approach and optimized performance for better user experience and client engagement.


---

## ✨ Features

- ⚡ Fast and optimized React frontend
- 🎨 Modern UI/UX design with smooth animations
- 📱 Fully responsive (mobile, tablet, desktop)
- 🧭 Multi-page routing system
- 🔥 Smooth scroll navigation between sections
- 🌙 Light/Dark mode support (if enabled)
- 📬 Contact form with email functionality
- 📧 Automated email system (user + admin notification)
- 🧠 SEO optimized using meta tags (Helmet)
- 🔗 Open Graph (OG) support for social sharing
- 💡 Clean and reusable component structure
- 🚀 Deployed on Vercel

---

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Framer Motion (animations)
- React Hook Form
- React Icons
- React Helmet (SEO management)
- Axios

### Backend
- Node.js
- Express.js
- Nodemailer (email sending system)
- CORS
- Dotenv

---

## 📩 Contact System

This project includes a fully functional contact system:

- Sends email to portfolio owner when a user submits a message
- Sends automatic reply email to the user
- Secure environment variables used for authentication
- Gmail SMTP integration via Nodemailer

---

## 📈 SEO Optimization

This project is optimized for search engines using:

- Dynamic page titles (React Helmet)
- Meta descriptions for each page
- Clean URL structure with React Router
